HOW TO INSTALL: QUICKIE-AUTOMATION
=========================================================================================

1. Quickie-Automation should be extracted inside the C: drive.
- why? because Quickie has been hard coded to work within the C: drive to any computer.
2. Find the Installation Script and run with powershell.
- location : "C:\Quickie-Automation\pow.scripts\QuickiePow\modules\install\installation.ps1"
- run installation.ps1 with powershell.
3. Quickie should be now sent to desktop and ready to be used.
4. done.
